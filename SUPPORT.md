# Getting Help

## Where to Ask

| What you need | Where to go |
|---|---|
| **Question about an extension** | The extension's GitHub Discussions, or its issue tracker if you've confirmed it's a bug |
| **Question about the SDK or Path Design System** | Ellucian's [Developer Community](https://github.com/ellucian-developer) — they maintain the platform |
| **General community Q&A** | [Org-level GitHub Discussions](https://github.com/orgs/experience-extension-community/discussions) |
| **Proposing a new extension** | Open an `Extension Proposal` issue in this `.github` repo |
| **Bug report** | The affected extension's issue tracker |
| **Feature request** | The affected extension's issue tracker |
| **Security vulnerability** | See [SECURITY.md](./SECURITY.md) — do not open a public issue |
| **Code of Conduct concern** | See [CODE_OF_CONDUCT.md](./CODE_OF_CONDUCT.md) for the reporting email |

## What This Project Does NOT Provide

- **Commercial support.** This is a volunteer community project. Institutions deploy these extensions at their own risk and are responsible for their own production support.
- **SLAs.** Maintainers respond on a best-effort basis. Critical bugs are typically addressed within days; feature requests can take much longer.
- **Custom development.** Maintainers are not available for paid customization work in their community-maintainer capacity.
- **Ellucian product support.** For questions about Ellucian Experience itself, the Path Design System, or Ethos APIs, consult Ellucian's official support channels.

## Helping Yourself First

Before asking, please:

1. Search closed issues and Discussions — your question may already have been answered.
2. Read the affected extension's `README.md` and `docs/` folder thoroughly.
3. Check the [master architecture document](./ARCHITECTURE.md) for community-wide conventions.

## Helping Each Other

This community runs on reciprocity. If you've gotten help here, consider:

- Answering a question in Discussions
- Reviewing a PR
- Improving documentation
- Submitting a fix you discovered

The single biggest factor in this community thriving is whether members help each other.
